# 开机自启

Windows：

  1. 在 任务计划程序 中，创建基本任务；
  2. 拷贝文件到自启动位置（C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp）。

Notice：自启动读取相对路径文件时，64 位操作系统寻找 . == C:/windows/sysWOW64 ，32 位操作系统寻找 . == C:/windows/systems32 。
原因似乎是：如果 B 程序用相对路径获取路径，然后让 A 程序调用 B 程序，那么这个 B 程序获取的就不是自己的路径了而是 A 程序的路径，而开机自启也是由 Windows 某个进程调用的。
